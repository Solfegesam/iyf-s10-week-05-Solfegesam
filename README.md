## 📘 Overview

This project demonstrates how to build interactive web applications using **JavaScript DOM manipulation** and **event handling**. It focuses on creating dynamic user interfaces that respond to real-time user input.

---

## 🎯 Learning Objectives

* Select and traverse DOM elements efficiently
* Manipulate content, attributes, and styles dynamically
* Handle user events (click, input, submit, keyboard)
* Build responsive and interactive web applications

---

## 🚀 Features

### 📝 Interactive To-Do List

* ➕ Add new tasks
* ✅ Mark tasks as complete
* ❌ Delete tasks
* 🔄 Real-time UI updates
* ⌨️ Keyboard support (e.g., press Enter to add task)

---

### 📚 DOM Manipulation (Lesson 9)

* Element selection (`getElementById`, `querySelector`)
* DOM traversal (parent, children, siblings)
* Content updates (`textContent`, `innerHTML`)
* Attribute & style manipulation

---

### ⚡ Event Handling (Lesson 10)

* Click events
* Input events
* Form submission
* Keyboard events
